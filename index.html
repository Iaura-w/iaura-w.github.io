GitHub Repositories

A Java application designed to retrieve and display information about a user's GitHub repositories, specifically non-fork repositories and their branches. It connects to the GitHub API to gather repository data such as the repository name, owner's login, and branch details (branch name and last commit SHA). The application temporarily stores this information in a MongoDB database for 1 hour to minimize redundant API calls and improve performance.

Technologies: Java, Spring Boot, MongoDB, JUnit, Mockito, MockWebServer, Docker Compose, REST API

Android Application for Identifying Instruments

An Android application designed to identify musical instruments based on recorded sounds. The app is built in Kotlin, while the server, implemented in Python, handles audio analysis using a machine learning model. Social Network Application

The Social Network Application is a web platform designed to enable users to create posts and interact with others through likes. Users can register, log in, and post updates. While initially planned to include features like comments and friends lists, the core functionalities focused on creating and liking posts. The backend is built using Java and Spring Boot for RESTful API services. Spring Security ensures user authentication and authorization, and MySQL is used for data storage, including user profiles and posts. Swagger was used for API documentation. Project development was managed using Git for version control and Jira for task organization and team collaboration.

Technologies: Java, Spring Boot, Spring Security, REST API, MySQL, Swagger, Jira
